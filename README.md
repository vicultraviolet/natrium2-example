# Natrium2 Example

## Introduction
Simple example application of the [Natrium2 Library](https://github.com/vicultraviolet/natrium2/)


## Cloning
This repository uses git submodules, so it is important to either clone it recursively:

```
git clone --recursive https://github.com/vicultraviolet/natrium2-example.git
```

Or get the submodules manually by running:

```
git submodule init
git submodule update
```
